# Array Problem Solutions

This repository contains a curated collection of array problems and solutions in Java, ranging from **easy** to **hard** levels. It is intended for learning, practice, and improving problem-solving skills, especially for coding interviews and competitive programming.

## 📂 Contents

- ✅ Array problem solutions implemented in Java
- 🧠 Covers a variety of concepts such as:
  - Sorting
  - Searching
  - Sliding window
  - Prefix sum
  - Two pointers
  - Hashing and more

> **Note:** Problems are not yet categorized by difficulty level.

## 🚀 Getting Started

1. Clone the repository: git clone https://github.com/meantaraa/array-problems-solutions.git
   

2. Open in any Java IDE (e.g., IntelliJ IDEA, Eclipse) or compile via terminal:
   ```bash
   javac YourFileName.java
   java YourFileName
   ```

## 🛠️ Requirements

- Java 8 or later
